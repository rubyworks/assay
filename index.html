Build Solid Assertion Frameworks

Assay defines a set of reusable assertion classes in the same way that Ruby defines exceptions.

Indeed, the Assertion base class is a subclass of Exception itself.

Assay provides a complete set of these assertion classes for the most common assertion needs.

Assertion nomenclatures can use Assay to provide a solid foundation for their assertion methods.

Assay can be used with TestUnit, MiniTest, RSpec and almost any other test harness.

Learn more by reading the API and QED documentation.

And whatever you do be sure to, "Test! Test! Test!"
